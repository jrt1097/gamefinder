# GameFinder

GameFinder is a cloud-based event discovery application that helps users find gaming, sports, and community events near them.

## Project Goal

The goal of this project is to build one integrated cloud-based application using:

- Cloud data storage
- Serverless compute with a REST API
- Content Delivery Network
- Deployment automation

## Main Features

- Browse events
- View nearby events
- Search and filter events
- View event details
- Future support for user signups or saved events

## Planned Cloud Architecture

- Frontend hosted as a static website
- CDN in front of the frontend
- REST API using serverless compute
- Cloud database or storage for event data
- Automated deployment through GitHub Actions
